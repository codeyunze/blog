正弦和余弦加法公式的推导

正弦和余弦的加法公式是三角学中最核心的恒等式之一，其标准形式为：

sin⁡(α+β)=sin⁡αcos⁡β+cos⁡αsin⁡βcos⁡(α+β)=cos⁡αcos⁡β−sin⁡αsin⁡β

以下是几种经典的推导方法：

方法一：几何法（单位圆与旋转）

核心思想：通过单位圆中角度的几何关系推导。

  1. 构造图形：

    • 在单位圆上，设点 ( P ) 对应角度 ( α )，点 ( Q ) 对应角度 ( α+β )。
    • 点 ( P ) 的坐标为 (cos⁡α,sin⁡α) ，点 ( Q ) 的坐标为 (cos⁡(α+β),sin⁡(α+β))。

  2. 旋转角度 $ \beta $：

    • 将向量 ( OP ) 旋转角度 (β) 得到向量 ( OQ )。

    • 旋转后的坐标变换公式为：

      {cos⁡(α+β)=cos⁡αcos⁡β−sin⁡αsin⁡β,sin⁡(α+β)=sin⁡αcos⁡β+cos⁡αsin⁡β.

    • 这直接给出了余弦和正弦的加法公式。

几何直观：旋转矩阵的乘法对应角度相加的三角函数值。

方法二：欧拉公式（复数法）

核心思想：利用复指数函数的性质。

  1. 欧拉公式：eiθ=cos⁡θ+isin⁡θ
  2. 指数相加：ei(α+β)=eiα⋅eiβ展开后：cos⁡(α+β)+isin⁡(α+β)=(cos⁡α+isin⁡α)(cos⁡β+isin⁡β)
  3. 展开乘法：=cos⁡αcos⁡β−sin⁡αsin⁡β+i(sin⁡αcos⁡β+cos⁡αsin⁡β)
  4. 实部与虚部分离：
    • 实部对应余弦公式：cos⁡(α+β)=cos⁡αcos⁡β−sin⁡αsin⁡β。
    • 虚部对应正弦公式：sin⁡(α+β)=sin⁡αcos⁡β+cos⁡αsin⁡β。

优势：简洁且无需几何构造，适用于复数领域。
